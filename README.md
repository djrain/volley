### Gradle with jitpack

#### Add it in your root build.gradle at the end of repositories:
```javascript

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

```
#### Add the dependency
```javascript

	dependencies {
	        compile 'com.github.djrain:volley:1.0.0'
	}


```
